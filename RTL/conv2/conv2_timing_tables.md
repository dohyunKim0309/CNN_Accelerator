# Conv2 Timing — Tables

`conv2_timing.md` 의 표만 발췌해 사람이 빠르게 훑어보기 위한 문서.

## 0. 약속 (최소)

- 셀 값 = **cycle 시작 시점의 register 값** (직전 edge 에서 latch 된 값).
- counter = `(row_cnt, col_cnt)`. 0 ≤ row_cnt ≤ 25, 0 ≤ col_cnt ≤ 25 (cap 보장).
- `(R, C)` = 입력 row R, col C 의 8 IC × 8b 데이터.
- BRAM L=2 (Primitive Output Register Enable). `ENA = REGCE = shift_en` (gating).
- `dout` = BRAM out reg 값 (= c1c2_dout).
- `win_r2[i]` = window row 2 (newest row, BRAM 직결 흐름), col i (i=0 oldest, i=2 newest). win_r1, win_r0 도 동일 패턴이며 각각 row 1, row 0.
- `col_sel`: PE input col selector. WRAP 에서 0 고정, 그 외엔 = `kw_cnt`.
- `K_col`: PE weight 선택 = `sel` = `kw_cnt`.
- `phase` 컬럼:
  - HOLD/ADV → `kw=N` (N=0/1/2)
  - WRAP → `w=N`
  - DRAIN → `d=N` (N=0..11)
  - FILL/IDLE/LOAD/DONE → `—`
- `PE in col` = `(win_r0[col_sel], win_r1[col_sel], win_r2[col_sel])` — PE 입력 3-tuple (oldest, mid, newest row).
- `Cnt → Cnt` = "이 cycle 의 counter → 다음 cycle 의 counter" (edge 결과).
- `sft` = shift_en. `—` = not applicable.
- Pipeline depth (참고): PE input @ T → c2pool memory updated @ edge (T+11→T+12). 총 12 cycle.

State 약어: FILL=PIPELINE_FILL, HOLD=COMPUTE_HOLD, ADV=COMPUTE_ADVANCE, WRAP=COMPUTE_WRAP, DRAIN, DONE.

Cycle 0 = PIPELINE_FILL 첫 cycle. counter @ 0 = (0, 0). cycle k 의 counter (FILL 동안) = `(k div 26, k mod 26)`.
